# wan27_org

Dart package for [wan27.org](https://wan27.org).

This package provides a lightweight Dart entry point for projects that need to
reference the wan27.org platform. It is intentionally small and ready to expand
into a fuller Dart or Flutter integration package.

## Installation

```bash
dart pub add wan27_org
```

## Usage

```dart
import 'package:wan27_org/wan27_org.dart';

void main() {
  print(hello());
  print(homepage);
}
```

## Included Utilities

- `homepage` pointing to `https://wan27.org`.
- `hello()` for confirming the package is installed.
- pub.dev metadata for homepage, documentation, repository, and issue tracker.
